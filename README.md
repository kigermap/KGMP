# 🎭 KGMP — kigurumi 烧鸡类型测试

> 戴上面具之后，你是哪种烧鸡？

纯前端趣味测试单页应用，从飞书多维表格动态拉取题目，随机抽取 15 题作答。

## 文件结构

```
kigmap-quiz/
├── index.html      # 入口页
├── style.css       # 样式
├── app.js          # 主逻辑（题目加载、答题、提交、结果）
├── data.js         # 离线兜底题库（飞书不可用时使用）
├── api/
│   └── index.js    # 阿里云 FC 代理函数（飞书 API CORS 代理）
└── README.md
```

## 业务流程

1. 从飞书「题库」表拉取启用状态的题目
2. 随机打乱，抽取 15 题
3. 用户逐题作答（A=1分 B=2分 C=3分 D=4分）
4. 分析动画 → 匹配「结果映射」表的分数区间
5. 展示结果（类型 + 描述 + 金句 + 同型占比）
6. 写入「提交记录」表（原始题号映射、来源检测、完成时长）
7. 支持复制分享 / 换题重测

## 部署

静态文件直接托管：GitHub Pages / Vercel / Netlify / 任何静态服务器。

需要先部署 `api/index.js` 到阿里云函数计算（FC Web 函数）作为飞书 API 代理。
